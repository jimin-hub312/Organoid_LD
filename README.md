Supplements for 3D Quantitative Analysis of Lipid Droplets   

Supporting codes and data for a paper titled “Label-free 4D analysis of lipid droplet in hepatic organoids”.

Instructions for use:
1) Download testing datasets:https://doi.org/10.5281/zenodo.18251200
2) In `main.m`, set `inMatPath` to the full path of the downloaded dataset file.
3) Run `main.m`
